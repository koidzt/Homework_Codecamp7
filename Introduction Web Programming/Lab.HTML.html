1. attribute

Tag Attribute Definition
h The h1-h6 tags also supports the Global Attributes in HTML. The h1-h6 tags are used to define HTML headings.
p The p tag also supports the Global Attributes in HTML. The p tag defines a paragraph.
div The div tag also supports the Global Attributes in HTML. The div tag defines a division or a section in an HTML document.
img The img tag has two required attributes: 1.src and 2.alt The img tag is used to embed an image in an HTML page.
a The a tag also supports the Global Attributes in HTML. The a tag defines a hyperlink, which is used to link from one page to another.
ul The ul tag also supports the Global Attributes in HTML. The ul tag defines an unordered (bulleted) list.
ol The ol tag also supports the Global Attributes in HTML. The ol tag defines an ordered list. An ordered list can be numerical or alphabetical.
table The table tag also supports the Global Attributes in HTML. The table tag defines an HTML table.
form The form tag also supports the Global Attributes in HTML. The form tag is used to create an HTML form for user input.
button The button tag also supports the Global Attributes in HTML. The button tag defines a clickable button.

3. input type

input type Definition
button Defines a clickable button (mostly used with a JavaScript to activate a script)
checkbox Defines a checkbox
color Defines a color picker
date Defines a date control (year, month, day (no time))
datetime-local Defines a date and time control (year, month, day, time (no timezone)
email Defines a field for an e-mail address
file Defines a file-select field and a "Browse" button (for file uploads)
hidden Defines a hidden input field
image Defines an image as the submit button
month Defines a month and year control (no timezone)
number Defines a field for entering a number
password Defines a password field
radio Defines a radio button
range Defines a range control (like a slider control)
reset Defines a reset button
search Defines a text field for entering a search string
submit Defines a submit button
tel Defines a field for entering a telephone number
text Default. Defines a single-line text field
time Defines a control for entering a time (no timezone)
url Defines a field for entering a URL
week Defines a week and year control (no timezone)
